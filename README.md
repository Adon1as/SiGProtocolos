# SiGProtocolos
Sistema de Gestão de Protocolos

Como Rodar:
  Com o Docker instaladado rode `docker-compose up --build` na raiz do projeto e acesse o link `http://localhost:8081/cadastrar-protocolos`.

Outras informações:  
    Java: 17  
    Backend:SpringBoot    
    Frontend:Vaadin  
    Banco: PostgreSQL
